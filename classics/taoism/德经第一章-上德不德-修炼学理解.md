## 一、原文（帛书乙本,公版）

> 上德不德,是以有德。下德不失德,是以無德。
> 上德無為而無以為也。上仁為之而無以為也。上○為之而有以為也。
> 上禮為之而莫之應也,則攘臂而乃之。
> 故失道而后德,失德而句（后）仁,失仁而句（后）義,失義而句（后）禮。

注:此即通行本第 38 章;帛书本德经在前,故为德经首章。甲本此章残损较重（前十六字全缺）,乙本完整,○为残一字（据文义为"義"）。乙本"句"是"后"的记音异写。值得注意的是,帛书两本均**没有**通行本"下德為之而有以為"一句,学界多认为该句是后世衍文。另:通行本"攘臂而扔之"处,帛书作"乃"。

## 二、关键字义（转述干昌新训释,逐条标出处）

干昌新（气功师,著有《破译〈老子〉祖本》《老子人体生命科学》《辅教170问答》等）读《老子》的总前提是:**这是一部人体生命科学著作**,"德、仁、义、礼、有、无、上、下"都是修炼术语,不是社会伦理概念（《破译》德经第 1 章点评）。以下均为**观点转述**,原文表述请查对应出处:

| 字 | 修炼学读法（转述） | 出处指引 |
|---|---|---|
| 上 | 人体巅顶,指大脑 | 《破译》德经第1章点评（约 p.242）;《老子人体生命科学》德经首章 |
| 下 | 人体底部——以会阴为基底的腹底三角区 | 同上 |
| 德 | 生命功能的哺育升登（取《说文解字注》"德,升也;升当作登"之字源） | 同上 |
| 不 | 语助词,无实义（非否定词） | 《破译》德经第1章点评 |
| 有 | 气功态下从自然界获得的特殊营养素、特殊能量及其功能转换（《说文注》"有,不宜有也"——常人不能有而气功态能有） | 同上;《生命科学》 |
| 无 | "无"的意识功能态,分层次;本章特指最高层次的"定而有识"态 | 同上 |
| 為 | 坐善修炼（繁体"爲"象母猴之坐形,喻打坐调形的姿势） | 《破译》道经第2章 |
| 以 | 用——"無以為/有以為"即用"无"/用"有"去修炼 | 《生命科学》德经首章 |
| 也 | 在"…為也"句式中依《说文》本义引申读作"阴性事物",充当修炼的宾语 | 《生命科学》德经第30、36章 |
| 仁 | 机体组织之间互相协调、亲密无间 | 《辅教170问答》第13问;《破译》德经第1章点评 |
| 義 | 各组织器官各司其职、各各相宜 | 同上 |
| 禮 | 对体质与大脑机能的规范改造 | 同上 |
| 失 | 释放（循《说文注》"捨→释→解"的引申链）,非"丧失" | 《生命科学》 |
| 而后 | 然后才能（时序连词） | 《生命科学》 |

## 三、逐句白话（本文作者按上表字义自行组织的表达）

**上德不德,是以有德。**
大脑（人体巅顶）意识层的高功能,是通过哺育先天潜能而升登起来的——这就是"有德":以"有"（特殊营养素、特殊能量及其功能转换）来完成这场哺育。（"不"是语助词,此句并非"上德不要德"。）

**下德不失德,是以無德。**
腹底（会阴一带）的生命功能升登,靠的是"释放"先天潜能——这就是"无德":在"定而有识"的意识功能态中完成哺育升登。

**上德無為而無以為也。**
巅顶层面的升登,要在"无"的状态下坐善修炼,并用这个"无"去修炼阴性事物。

**上仁為之而無以為也。**
让大脑各组织亲密无间（仁）,坐善为之,同样用"无"来修。

**上義為之而有以為也。**
让各组织各司其职、各得其宜（义）,坐善为之,这一步用"有"来修。

**上禮為之而莫之應也,則攘臂而乃之。**
对体质与大脑机能做规范改造（礼）,坐善为之却一时没有回应——那就伸出手臂,像对着"你"一样直呼直唤它。（"乃"训"你";这句描绘规范改造初期身体无应和时的功法情境。）

**故失道而后德,失德而后仁,失仁而后義,失義而后禮。**
先释放"道"（功能转换）,"德"（哺育升登）才能随之而来;释放了德,组织间的亲密（仁）才能出现;释放了仁,各得其宜（义）才能成立;释放了义,体质的规范改造（礼）才能完成。（"失"训"释放"、"而后"训"然后才能",所以这四句不是"道德沦丧的退化史",而是一条修炼的**释放次第**——与通行本的读法方向完全相反。）

## 四、全章义理串讲（本文归纳）

按这套读法,德经首章讲的是**开发大脑意识层高功能的两个基本条件**:

1. **"有"与"无"的结合**——"有"是物质面（特殊营养素、能量及其转换）,"无"是意识面（"定而有识"的功能态）。"有德"与"无德"不是好坏对立,而是同一过程的两个支撑。
2. **"上"与"下"的结合**——巅顶（大脑）与腹底（会阴区）是两个重点修炼体位,潜能在两地之间被哺育、规范、升达。

然后用"德→仁→义→礼"给出**修炼的递进次第**:功能转换先行,哺育升登随后,再到组织协调（仁）、各得其宜（义）、体质规范（礼）——每一步都以"释放"前一步为前提。

这套读法最与众不同的三个点:"不"是语助（全章没有一处否定德）、"失"是释放（不是丧失）、"也"是阴性事物（不是语气词）——三个虚字的重训,把一章"伦理退化论"整个翻转成了"修炼次第论"。

## 五、实践对照（Layer 4 · 仅供现代读者理解参考,非原文含义）

- **与静坐实践**:此章"無為而無以為"的读法（在定境中修、用定境修）与静坐传统里"不是不做,而是在定中做"的经验描述方向一致。对照供理解,两个体系的术语并不互训。
- **与辟谷实践**:"有=特殊营养素"的训释,正是气功辟谷"不食五谷而气足神旺"的理论出口——气功态下的能量获取渠道不同于日常饮食。本仓辟谷主题（见主 README）可与此互参。
- **与通行读法**:通行本把此章读为"上德之人不自恃有德,所以才有德"——把"不德"读作"不执著于德"。修炼学读法与它殊途:一个在心性层面讲不执著,一个在生理层面讲哺育升登。两个读法不需要分对错,放在一起恰好看清各自的立足点。
