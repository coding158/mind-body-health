## 一、原文（帛书乙本，公版）

甲本此段几乎全残（连续四十余个缺字符 □，仅"道善"二字尚可辨），以乙本为底：

> 上德如浴，大白如辱，廣德如不足，建德如□，質□□□，大方無禺，大器免成，大音希聲，天象無刑，道褒無名。‧夫唯道，善始且善成。

注：此即通行本第 41 章后段（其前段"上士闻道"即本仓第五章「三士三位」）。干昌新的德经分章依帛书原序，本章在其分章中列**第六章**。乙本残缺处，干氏勘定补作"建德如輸""質真如渝"；乙本"道褒無名"干氏勘定作"道隱无名"（褒/隱两读并存，待图版终审）。甲本仅存的"道善"二字，恰落在勘定"夫唯道，善始且善成"的位置上——成了干氏此处分章的一个内部旁证。通行本此段改"如"为"若"、改"免"为"晚"、改"禺"为"隅"，末句作"善貸且成"。

## 二、关键字义（转述干昌新训释，逐条标出处）

干昌新读此章的总纲是：全章用五个"随从（如）"，对应潜能开发上要落实的**五件事**——它不是"上等的德好像空谷"那类德性比喻，而是一套**功法次第**。以下均为**观点转述**：

| 字     | 修炼学读法（转述）                                                                | 出处指引                                |
| ----- | ------------------------------------------------------------------------ | ----------------------------------- |
| 如     | 随从——本义跟随。全章"某德如某事"是"德随从着某事"的功法句式（通行本把十个"如"全改成"若"，功法语气随之丢失）               | 《破译》p.246 点评                        |
| 浴     | 净化意识——"上德如浴"即巅顶大脑之德随从着意识的净化（通行本改为"谷"）                                    | 《破译》pp.163、246；**《辅教170问答》第126问侧证** |
| 辱     | 除去贪欲杂念——"大白如辱"随从着对贪念的荡涤                                                  | 《破译》p.246；**《辅教170问答》第126问侧证**      |
| 不足    | 功能转换的修炼始终处在"未自满、未止步"的进行状态（"廣德如不足"对应"开展功能转换"一事）                           | 本文按点评语境推演，信心 low                    |
| 輸     | 使先天潜能长大成熟、结成正果（"建德如輸"）                                                   | 《破译》p.246；**《辅教170问答》第126问侧证**      |
| 質 / 真 | 質＝亏欠的体质；真＝使体质得到变化改造（"質真如渝"一句讲的是改造体质）                                     | 《破译》p.246；**《辅教170问答》第126问侧证**      |
| 禺     | 母猴，与"爲"同类，都指修道时的坐善调形（通行本改为"隅"角落，两字义完全不同）                                 | 《破译》p.164 注疏                        |
| 免     | 古同"兔"，取《说文》"兔，逸也，从免不见足"——逸然不见其足迹（通行本改为"晚"；故"大器免成"不是"大器晚成"，而是"大器逸然不见迹而成"） | 《破译》p.164 注疏（引《说文》）                 |
| 刑     | 本义戮颈，引申为"割离"——"天象无刑"即图景映现时不带割离的痕迹（通行本改为"形"）                              | 《破译》p.164 注疏；道经相应章"刑＝割离"既有成例        |
| 隱     | 人体隐蔽之处——功能转换藏在身体的隐蔽部位、运行不停，故此句用"隱"字来说它                                   | 《破译》p.246 点评                        |
| 名     | 自我呼唤（道经首章"名可名也"既有成例）——"无名"不是没有名声，而是功能转换不以"自我呼唤"的方式显现                     | 本文按既有成例推演，信心 med                    |
| 善     | 特殊营养——"善始且善成"即以获得特殊营养开始、并以之建功立业                                          | 道经相应章既有成例；《破译》p.246 点评              |

（"上＝巅顶大脑""德＝哺育升登""大＝神通广大""道＝功能转换"沿用本系列前几篇已建立的专名系统，此处不再展开。）

## 三、逐句白话（本文作者按上表字义自行组织的表达）

**上德如浴。** 巅顶大脑之德，随从着"浴"——意识的净化。

**大白如辱。** 广大而明澈的潜能之明，随从着"辱"——对贪欲杂念的荡涤。

**廣德如不足。** 广大的德，随从着"不足"——功能转换的修炼始终在未自满、未止步的进行之中。

**建德如輸。** 树立起来的德，随从着"輸"——使先天潜能长大成熟、结成正果。

**質真如渝。** 亏欠的体质（質）得到变化改造（真），随从着"渝"——渝变。

**大方无禺，大器免成，大音希声，天象无刑。** 大方正没有"禺"（坐善调形的痕迹），大器"免"（如兔逸去、不留足迹）而成，大音深妙、罕闻其声，天象（图景映现）没有"刑"（割离之迹）。

**道隱无名。** 功能转换隐于人体隐蔽之处、不以"自我呼唤"的方式显现，故曰"无名"。

**夫唯道，善始且善成。** 唯有功能转换：以获得特殊营养（善）开始，也以获得特殊营养而成就。

## 四、全章义理串讲（本文归纳）

在修炼学读法里，本章是一份**功法次第清单**。五个"如（随从）"把要做的几件事逐一排开：从清理意识（浴）、荡涤贪欲（辱），到把功能转换持续推进（不足／道）、让先天潜能长成正果（輸），再到改造亏欠的体质（質真）。它和本系列前几篇——首章的"上德／下德"、第三章的"天地神"生理映射——连起来，构成一条从清理、到转化、再到固化的完整链条。

本章最值得一提的，是它有**两部书互证**。《破译》第 246 页的点评，把这五个"如"讲成五件必须做的事；而《辅教170问答》第 126 问在回答"开发人体潜能要做好哪几件事"时，列出的五件与之**逐条对应**。同一套读法在作者两部不同著作里各自独立出现，说明它不是对某一句的临时发挥，而是这个体系里稳定的功法框架。

几个被通行本"改笔"盖住的细节，正是这一读法的支点：

- **"如"不是"若"。** 通行本把十个"如"全改成"若"，"上德若谷"式的比喻由此成形；而"如"的本义是"随从"，"上德如浴"讲的是"德随从着净化意识"这个功法动作，不是"上德好像山谷"的形象比喻。
- **"大器免成"不是"大器晚成"。** "免"古同"兔"，《说文》说"兔，逸也，从免不见足"——照这个字义，"免成"是"像兔子般逸去、不留足迹地成就"，与励志格言"大器晚成"（大才要长时间才成就）完全不是一回事。
- **"大方无禺"不是"大方无隅"。** "禺"是母猴，与"爲"同类，指坐善调形；通行本改成"隅"（角落），"大方无隅"（最大的方没有棱角）遂成一句几何式的玄言。

本章的推演成分很少——只有"不足"和"名"两字属本文按语境推演（标信心 med／low），其余各字都有出处。甲本此段几乎全残，只"道善"二字尚可辨认，恰好落在"夫唯道，善始且善成"的位置上，成了干氏此处分章的一个巧合旁证。

## 五、实践对照（Layer 4 · 仅供现代读者理解参考，非原文含义）

- **与"功法次第"类实践**：这一串"先清理、后转化、再固化"的安排，与不少传统修炼体系"先净念、再入定、后转化"的次第结构相似——都把"清理在前、转化在后"当作基本顺序。此处仅作结构类比，不等同于原文含义。
- **与"大器晚成"的世俗读法对照**：通行本"大器晚成"两千年来被读作励志格言（大才需要时间打磨）。修炼学读法回到帛书"大器免成"，把它读成"大器不见其迹而成"的功境描述——两种读法指向完全不同的话语世界，恰好说明"一个字的版本差异，可以改变整句所在的世界"。
- **与"净念为先"的静心经验对照**："上德如浴＝随从着意识的净化"，与冥想／静心传统中"先清理杂念"的入门步骤有可比之处；把"净化意识"排在几件事的第一位，也与"净念为先"的通行经验相合。此为理解参考，非原文断言。
