## 一、原文（帛书甲/乙本，公版）

甲本此段前有八字残缺，乙本完整：

> 夫禮者，忠信之泊也，而亂之首也。
> 前識者，道之華也，而愚之首也。
> 是以大丈夫居其厚，而不居其泊；
> 居其實，不居其華。
> 故去皮取此。

注：此即通行本第 38 章后段；帛书本德经在前，故为德经第二章。甲本"去皮取此"处，乙本作"去罷而取此"，通行本作"去彼取此"——三本尾句各不相同，是本章最大的文本悬案（详见下文）。

## 二、关键字义（转述干昌新训释，逐条标出处）

干昌新读此章的总前提是：**忠、信、愚、厚四字全是修炼态度的专名，与社会伦理无涉**（《破译》德经第 2 章点评）。以下均为**观点转述**，原文表述请查对应出处：

| 字 | 修炼学读法（转述） | 出处指引 |
|---|---|---|
| 禮 | 对体质与大脑机能的规范改造（同德经首章"上禮"之训） | 《破译》德经第1章点评 |
| 忠 | 尽心——修炼须全身心投入 | 《破译》德经第2章点评（约 p.243） |
| 信 | 诚信——修炼须诚实不欺地面对自己的身心反应 | 同上 |
| 泊 | 停泊、靠岸不前（非通行本改写的"薄"） | 《破译》p.161 注疏 |
| 亂 | 治理——取《说文》"乱，治也"的古义，非混乱 | 道经相应章既有成例 |
| 前識 | 预先的识见，即"就事论事的聪明预判"（两字分读，非一词） | 《破译》p.243 点评语境 |
| 道 | 体内的功能转换（贯通全经的核心术语） | 道经首章成例 |
| 華 | 外发荣华——刻意追求时显露的表面相状（取《说文》"华，荣也"） | 本文作者按点评语境推演 |
| 愚 | 憨厚执著——修炼需要的一种"不耍聪明、老老实实"的态度 | 《破译》p.243 点评 |
| 大丈夫 | 神通广大的修炼者（"大"=人体高功能的形容词；"丈夫"=成年而能担当者） | "大"=道经既有条目；"丈夫"=本文推演 |
| 居 | 坐善——《说文》"居，蹲也"（俗字作踞，即坐），引申为坐善安处 | 道经既有成例 |
| 厚 | 反反复复地投入——集中精力和时间，不断积累修炼的厚度 | 《破译》p.243 点评 |
| 實 | 实腹——特殊能量逐步充实、积贮于腹腔中心 | 道经既有条目 |
| 去 | 去除、分离 | 道经既有条目 |
| 罷 | 偷惰之人（取《说文注》"罷民、罷士，谓偷惰之人"） | 《破译》p.161 勘误主张（待图版确认） |
| 耳 | 急言语词（取《说文注》"急言之曰耳"） | 同上 |
| 此 | 基础、支柱——指代居厚居实这一根本 | 道经既有条目 |

## 三、逐句白话（本文作者按上表字义自行组织的表达）

**夫禮者，忠信之泊也，而亂之首也。**
礼——对体质的规范改造这一层——是尽心与诚信的停泊处（靠岸不前）。而这正是"治理"（身体的整顿与开发）的开端。（"亂"取古义"治理"，全句不是"礼是混乱的开端"，而是"礼这一步是整顿修炼的起点"。）

**前識者，道之華也，而愚之首也。**
预先的识见——就事论事的聪明预判——是功能转换的外发荣华（刻意追求之相）。而憨厚执著，才是它的真正起点。（真功夫以"愚"为起点，不以聪明预判为起点。）

**是以大丈夫居其厚，而不居其泊；**
因此，神通广大的修炼者，坐善于反反复复投入的积厚，不坐善于停泊不前；

**居其實而不居其華。**
坐善于实腹（特殊能量的充实），不坐善于外发的荣华（表面的光鲜）。

**故去罷耳此。**
所以，去除偷惰（"罷"），急急于此（"耳此"）——急急于这座基础支柱。（尾句三本各不相同：甲本释文作"去皮取此"、乙本释文作"去罷而取此"、干氏勘定作"去罷耳此"。干氏认为甲本释文"皮""取"分别是"罷""耳"的误认，乙本"罷"字为这一主张提供了半个旁证。本文暂依干氏勘误主张呈现，终审须核对帛书原件图版。）

## 四、全章义理串讲（本文归纳）

此章接在德经首章"上德不德……失道而后德，失德而后仁，失仁而后義，失義而后禮"之后，专门讲**修炼者面对"礼"这一层时应该取什么态度**。

干氏读法的核心洞察是：忠、信、愚、厚这四个字，在社会语境里指"忠心""诚信""愚笨""厚道"，但在本章里全是**功夫态度的技术术语**——忠是尽心投入，信是诚实面对身心反应，愚是不耍聪明的憨厚执著，厚是反复积累不偷懒。

本章的论证结构很清晰：先指出"礼"（规范体质）和"前识"（聪明预判）各自的局限——它们只是起点，不是终点；然后给出修炼者的正确选择：居厚（反复投入）不居泊（浅尝辄止），居实（实腹）不居华（表面光鲜）；最后以一个斩钉截铁的"故"收束——去除偷惰，急急于此。

与通行本最大的三个分歧：（1）"泊"不是"薄"——干氏读"停泊"（靠岸不前），通行本改作"薄"（浇薄），方向一致但训释路径不同；（2）"首"不是"始"——通行本把"亂之首""愚之首"改成了"亂之始""愚之始"，干氏认为原文就是"首"，义为开端而非初始；（3）尾句"去罷耳此"——干氏认为帛书原件写的是"罷"（偷惰）和"耳"（急言语词），不是"彼"和"取"，这是针对帛书整理小组释文的一项勘误主张，裁决权在图版。

## 五、实践对照（Layer 4 · 仅供现代读者理解参考，非原文含义）

- **与静坐实践**：此章"居厚不居泊"——反复投入、不浅尝辄止——正是静坐传统中"功夫靠时间堆出来"的经验总结。两个体系用语不同，道理相通。
- **与日常修炼态度**："愚"字在此读作憨厚执著（褒义），而非愚蠢（贬义），这与许多修炼传统里"大智若愚"的阶段描述方向一致——到了一定阶段，聪明反而碍事，老老实实才是最有效的。
- **与通行读法**：通行本把此章读为"大丈夫处其厚，不居其薄；处其实，不居其华。故去彼取此"——核心义涵（取厚去薄、取实去华）与修炼学读法大致同向，但细节训释不同。两读并观，可各见其立足点。
