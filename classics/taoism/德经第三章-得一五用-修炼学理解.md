## 一、原文（帛书甲/乙本，公版）

甲本有少量残缺（□），乙本完整：

> 昔之得一者，天得一以清，地得□以寧，神得一以霝，浴得一以盈，侯□□□而以為正。
> 亓致之也：胃天毌已清將恐□，胃地毌□□將恐□，胃神毌已霝將恐歇，胃浴毌已盈將恐渴，胃侯王毌已貴□□□□□。
> 故必貴而以賤為本，必高矣而以下為基。

注：此即通行本第 39 章前段；帛书本德经在前。通行本在此段之后擅自增入"万物得一以生""万物无以生将恐灭"两句——甲乙两本均无此两句，且与甲本残缺字数的验证相合（残缺处的字数恰好容不下增入的两句）。

## 二、关键字义（转述干昌新训释，逐条标出处）

干昌新读此章的总前提是：此章讲的是**"定而有识"功能态对开发人体潜能的五个方面的作用**，文中的"天、地、神、浴、侯王"全部是人体修炼术语（《破译》德经第 3 章点评，约 pp.243-244；《老子人体生命科学》散见引用互证）。以下均为**观点转述**：

| 字 | 修炼学读法（转述） | 出处指引 |
|---|---|---|
| 得一 | 获得"定而有识"的功能态——"一"指功能/能量 | 《破译》p.244 五用点评；道经首章既有成例 |
| 天 | 意识——人体修炼语境中"天"非自然界之天，而是意识的代称 | 本文按点评第一作用（意识层面的洁净清明）推论 |
| 清 | 洁净清明——意识层面的澄明状态（五用之一） | 《破译》p.244 第一作用 |
| 地 | 腹底部位——"地"非大地，而是人体会阴一带的腹底 | 本文按点评第二作用（腹底部位的保护安宁）推论 |
| 寧 | 保护安宁——腹底在功能态中获得安定（五用之二） | 《破译》p.244 第二作用 |
| 神 | 鼻息——"神"在本文非神灵，而是呼吸的代称 | 本文按点评第三作用（鼻息的止歇现象）推论 |
| 霝 | 零态——"霝"是"零"的本字，指意识领域处于无极化的零态（非通行本改写的"靈/灵"，两字意义完全不同） | 《破译》p.162 注疏 |
| 浴 | 净化意识——"浴"本义洗澡，应用于精神意识领域即净化意识（非通行本改写的"谷"） | 同上 |
| 盈 | 盈满——净化意识使特殊能量及功能转换充满全身（五用之四） | 《破译》p.244 第四作用 |
| 侯王 | 气功修炼者的大脑（"侯"=修炼者，"王"=大脑——两字复合指大脑意识层的主宰位） | 《生命科学》德经第3章同词训释 |
| 爲 | 坐善修炼——繁体"爲"象母猴坐形，喻打坐调形的姿势 | 道经相应章既有成例 |
| 天下 | 人体巅顶与腹底——"天"指巅顶大脑，"下"指腹底 | 道经相应章既有成例 |
| 正 | 规范——大脑和腹底得到规范（五用之五） | 《破译》p.244 第五作用 |
| 毋 | 通"无"——指"定而有识"的功能态，原文不可更改（通行本改"已"为"以"并失此义） | 《破译》p.162 注疏 + 《生命科学》多源互证 |
| 己 | 自己——释文"已"更还为"己"（字形更还，非义改） | 《破译》p.162 注疏 |
| 胃 | 说——古"胃""謂"不同义（取"胃"之"谷府"义引申为说） | 道经首章成例 |
| 歇 | 鼻息止歇——完全体呼吸下的特殊生命现象：肺鼻呼吸暂停，由穴位体呼吸取而代之 | 《生命科学》德经第3章原句引用 |
| 蹶 | 不动不朽——完全穴位体呼吸进入高级阶段时的稳固状态 | 同上 |
| 貴/賤 | 大脑（贵）与潜能（贱）——功能运动自下而上，腹底之"贱"正是大脑之"贵"的本基 | 《破译》p.244 评说 |
| 高/下 | 大脑之高以腹底之下为基础——与"贵以贱为本"互文 | 同上 |

## 三、逐句白话（本文作者按上表字义自行组织的表达）

**天得一以清。**
意识（人体巅顶）获得"定而有识"的功能态，便得以洁净清明。

**地得一以寧。**
腹底部位获得此功能态，便得以保护安宁。

**神得一以霝。**
鼻息获得此功能态，便进入"零态"——在定而有识中发生鼻息的止歇。

**浴得一以盈。**
净化意识获得此功能态，特殊能量及其功能转换便盈满全身。

**侯王得一以爲天下正。**
大脑（气功修炼者的主宰位）获得此功能态，通过坐善修炼（"爲"），使人体巅顶与腹底（"天下"）得到规范。

**其至之也：天毋己清將恐裂。**
推而言之：意识若非在"无"（定而有识）的功能态中自己清明，将恐涣裂。

**胃地毋己寧將恐發。**
（说）腹底若非在"无"态中自安，将恐发泄外散。

**胃神毋己霝將恐歇。**
（说）鼻息若非在"无"态中自处零态——但这里恰恰是"將恐歇"（将要止歇），而"歇"在干氏的体系里正是完全体呼吸的标志性现象。换言之，这一句不是警告，而是**描述功法效果**：进入定而有识后，鼻息将自然止歇，转为穴位体呼吸。

**胃浴毋己盈將恐渴。**
（说）净化意识若非在"无"态中盈满，将恐枯渴匮乏。

**胃侯王毋己貴以高將恐蹶。**
（说）大脑若非在"无"态中自贵自高，将恐颠蹶失稳——反过来说，"蹶"（不动不朽）恰是完全穴位体呼吸高级阶段的正面描述：大脑在定而有识中达到不动不朽的稳固状态。

**故必貴而以賤爲本，必高矣而以下爲基。**
所以，大脑之"贵"必以潜能之"贱"为根本（通过坐善修炼来达成），大脑之"高"必以腹底之"下"为基础。

## 四、全章义理串讲（本文归纳）

本章是德经篇中"专词系统"（天=意识/地=腹底/神=鼻息/霝=零态/浴=净化意识）的第一次成批展开。干氏读法的核心操作是：把通行本里看起来像宇宙论的陈述，全部**还原到人体内部**。

五句正面陈述（天清/地宁/神灵/浴盈/侯王正）→ 五句反面推展（恐裂/恐发/恐歇/恐渴/恐蹶）→ 两句总结（贵贱高下），结构严整对称。"毋"字是关键——干氏读"毋"通"无"，指"定而有识"的功能态，不是否定词"不要"。这样一来，后半段就不是"警告你不要如何如何"，而是"若非在无的功能态中自然如此，就会如何如何"——是条件推演，不是道德训诫。

本章最重要的发现是：**五用的"反面"恰好是功法效果的正面**。"歇"（鼻息止歇）和"蹶"（不动不朽）正是干氏体系中完全穴位体呼吸的两个标志性阶段，在《老子人体生命科学》中有完整的专节论述。这为"五用"与"五脏五行"的传统附会提供了截然不同的解释路径。

通行本在此章擅自增入"万物得一以生""万物无以生将恐灭"两句，把人体五用扩散成宇宙泛论，甲乙两本的残缺字数为此提供了硬证据——96 字的篇幅，无处容那 8 个字。

## 五、实践对照（Layer 4 · 仅供现代读者理解参考，非原文含义）

- **与静坐实践**："天清地宁"——意识清明、身体安宁——正是静坐入门时最基本的两个体征。本章把这两个常见的体验描述，放入了一个有严谨因果链条的解释框架（得一→天清/地宁/神灵/浴盈/侯王正）。
- **与辟谷/气功实践**："浴得一以盈"——净化意识使能量盈满——与气功辟谷中"不食五谷而气足神旺"的经验描述方向一致。
- **与通行读法**：通行本将"天得一以清"读为"天得到道就清明"，将"神"读为神灵——是宇宙论读法。修炼学读法将其全部落回人体，是人体生命科学读法。两套框架各有完整的自洽性，放在一起恰好看清各自的立足点。
