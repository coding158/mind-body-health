## 一、原文（帛书甲/乙本，公版）

甲本有残缺，乙本较完整。此处综合两本呈现（□为甲本残缺字）：

> 夫是以侯王自胃□孤寡不穀，
> 此亓賤□□與，非□，故致數與無與。
> 是故不欲□□若玉，硌﹦□□。

乙本：

> 夫是以侯王自胃孤寡不穀，此亓賤之本與，非也，故至數輿無輿。是故不欲祿﹦若玉，硌﹦若石。

注：此即通行本第 39 章后段。甲本"穀"字处，干昌新主张帛书原件是一个自创字（从土从一从禾，读 gǔ，义为人体摄入特殊营养素和特殊能量）——即其"老子自创二十九汉字表"的序 17。释文迳作"穀"，是整理小组的释读。这一分歧唯有核对帛书原件图版才能裁决。

## 二、关键字义（转述干昌新训释，逐条标出处）

本章核心句"此其以賤之本與非也故致數與无與"在《老子人体生命科学》中有完整的逐字解密——这是德经篇中**实录密度最高的章节之一**。以下均为**观点转述**：

| 字 | 修炼学读法（转述） | 出处指引 |
|---|---|---|
| 侯王 | 气功修炼者的大脑（"侯"=修炼者，"王"=大脑） | 《生命科学》德经第3章同词训释，多章一致 |
| 胃 | 说——古"胃""謂"不同义 | 《破译》p.162 注疏 + 道经首章成例 |
| 孤寡不槖 | 自谦之词——"孤"（孤单）、"寡"（寡少）、"不槖"（未得特殊营养素与能量之滋养），大脑以此自处，正是"以贱为本"的体现 | 《破译》pp.244-245 点评语境 |
| 槖 | 干氏主张的"自创字"（序17）——从土从一从禾：土=人体，一=特殊能量，禾=特殊营养素（谷物总称），三部件合起来表示"人体摄入特殊营养素与特殊能量"。释文写作"穀" | 《破译》pp.162-163 注疏 + 29字表 |
| 此 | 相比次而至——潜能一个接一个、依序开发，终至于止于至善（取《说文注》"此，从止匕。相比次而止也"） | 《生命科学》德经第4章逐字解密 |
| 賤 | 先天潜能的谦词——潜能本遭人轻贱，被视为无用之物，却正是"颊侧上出"的根柢 | 同上 |
| 本 | 根柢、根本 | 同上 |
| 與 | 施予且众多——"與"和"与"古义不同：都有施予的意思，但"與"含"多"的意义（朋群、共举），要将特殊营养素**众多地**施予潜能 | 同上 |
| 非 | 相背、剥离——取《说文》"非，韦也。韦者相背也"的本义。指意识层的高功能与孕育它的潜能组织彼此背离而分开——潜能一旦成熟,便像瓜熟蒂落一样脱出 | 同上 |
| 也 | 阴性事物——取《说文》"也，女阴也"的本义引申。指"相背"的特殊生命现象在阴性事物（人体潜能组织）中产生 | 同上 |
| 致 | 迎候宾客——取《说文》"致，送诣也"及段注"谓宾客所之也"。指在潜能开发成果（"宾客"）到来之处迎候 | 同上 |
| 數 | 疾速——取《说文注》"数……大约速与密二义可包之"。要迅速将特殊营养素施予潜能 | 同上 |
| 无 | "定而有识"的功能态 | 同上；道经既有成例 |
| 碌碌 | 玉之貌——潜能开发后如美玉般光华璀璨 | 《破译》p.245 点评 |
| 珞珞 | 石之貌——潜能坚实磊落如硕大山石 | 同上 |
| 不欲 | 不起贪欲——不贪求外在赞誉，因潜能本如玉石之贵 | 本文按点评"玉石喻潜能"推论 |

## 三、逐句白话（本文作者按上表字义自行组织的表达）

**夫是以侯王自胃，孤寡不槖。**
气功修炼者的大脑，自称"孤""寡""不槖"——以孤单、寡少、未得滋养自处。（这正是上一章"贵以贱为本"的落实：大脑虽为百脉所归之主宰位，却以卑下自居。）

**此其以賤之本與。**
这相比次而至的意识层高功能，其向上升"出"的根柢，正是人所轻贱的先天潜能。（"與"读"施予且众多"，此句是陈述，不是疑问——帛书本无问号，通行读法把它当反问句是另一条路径。）

**非也，故致數與无與。**
"相背"（剥离）的特殊生命现象，在阴性事物（潜能组织）中产生了。所以，要在宾客所至之处迎候"宾客"（潜能开发成果）的到来——迅速地、众多地把特殊营养素施予潜能，并把"定而有识"的功能态众多地施予潜能。

**是故不欲。**
所以不起贪欲——不贪求外在的赞誉与认可。

**碌碌若玉，珞珞若石。**
（潜能）如美玉般光华璀璨，如硕大山石般坚实磊落。

## 四、全章义理串讲（本文归纳）

本章上承第三章"贵以贱为本、高以下为基"，把它落实为一个具体的操作链：**营养施予 → 剥离 → 迎致**。

最核心的是那个设问句。"此其以賤之本與？"——大脑的高功能，是以卑贱的潜能为根柢吗？干氏的答案不是简单的"是"，而是用"非也"引出一个更精确的过程描述："非"不是否定词"不是"，而是《说文》本义"相背"——潜能组织与意识层高功能之间发生剥离，前者像瓜熟蒂落一样从后者脱离出来。然后才是"致"（迎候宾客）：剥离之后，在宾客（潜能开发成果）到来之处迎候。而整个过程的方法论是"數與无與"——疾速地、众多地施予（营养素），且在"定而有识"的功能态中施予。

这个"营养施予→剥离→迎致"的三段式，是该体系描述大脑意识层高功能开发的最精细段落之一。本章的逐字训释——此/賤/本/與/非/也/致/數/无——在《老子人体生命科学》中均有完整的《说文》字源链与气功专业释义，实录密度在德经篇中仅次于首章。

"不欲"（不起贪欲）的读法需要在全章语境中理解：潜能本如玉石——碌碌（玉之貌）与珞珞（石之貌）——自身就是至宝。既然已经是至宝，对外在赞誉的贪求就毫无必要。这是从"剥离—迎致"的功法操作自然推导出的修炼态度。

## 五、实践对照（Layer 4 · 仅供现代读者理解参考，非原文含义）

- **与静坐/气功实践**："剥离"概念在干氏体系中是一个专门术语——《辅教170问答》第72问有专题详解：潜能从成长→瓜熟蒂落→与潜能组织脱离。这与一些静坐传统中描述的"身心分离""气在外走"等体验有可比性，但各自术语体系的精确含义不宜互相代入。
- **与"不自满"的修炼传统**："侯王自称孤寡不槖"——大脑以卑下自居——与许多修炼传统中"谦卑是功夫的前提"的告诫方向一致。干氏的特殊之处在于给了它一个生理学解释：不是道德上的谦逊，而是功能态的自然需要。
- **与通行读法**：通行本的核心分歧在"致数誉无誉"——把"與"读作"誉"，于是整句变成"追求过多的荣誉反而没有荣誉"的道德格言。干氏的读法把它从道德劝诫变成了功法操作链，争议点在于"與/誉"的训释以及"非=相背"（vs"非=不是"）的《说文》本义取径。
