# 实践记录模板 · Practice Record Template

> 这里没有标准答案，只有真实的路。
> 你的记录不需要完整，不需要成功，走偏的部分同样有价值。

---

## 如何贡献你的实践记录

### 方式一：文字记录（直接提 PR）

复制以下模板，填写后保存为 `practice-records/你的GitHub用户名.md`，提交 PR 即可。

```markdown
## 基本信息

- 贡献者：（GitHub 用户名或匿名）
- 记录时间：
- 实践时长：（例：3个月 / 持续中）

## 实践阶段

- [ ] 食疗调整
- [ ] 素食
- [ ] 辟谷
- [ ] 观六触练习
- [ ] 转触练习
- [ ] 24小时静心尝试
- [ ] 其他：___

## 身体层面

**做了什么：**
（具体做法，不需要完整，写你真实在做的）

**身体的真实反应：**
（包括不适、改善、没有变化都可以写）

**目前卡住的地方：**

## 心理层面

**最容易被带走的触：**
- [ ] 眼触
- [ ] 耳触
- [ ] 鼻触
- [ ] 舌触
- [ ] 身触
- [ ] 意触

**有没有转触成功的瞬间：**
（描述那个瞬间是什么感觉）

**走偏过的地方：**
（这部分往往最有参考价值）

## 想问社区的问题

（你走到这里最想得到反馈的是什么？）

## 参考的经典或资料

（哪段原文对你帮助最大？）
```

### 方式二：录音记录

有时候说比写更自然，欢迎用录音分享你的实践过程。

**格式要求：**
- 时长：5～30分钟为佳
- 格式：MP3 / M4A / WAV 均可
- 语言：中文或英文

**提交方式（两种选其一）：**

1. **上传到网盘分享链接**
   百度网盘 / 阿里云盘 / Google Drive，把链接贴到 Discussions 里即可

2. **直接发到 Discussions**
   新建一个 Discussion，标题格式：`[录音] 你的主题`
   描述里附上链接和简短说明

**录音内容建议（不需要全部回答）：**
- 你是怎么开始这条路的？
- 身体或心理上有没有某个真实的转折点？
- 最难过的是哪一关？
- 现在是什么状态？

---

### 方式三：视频记录

记录日常实践过程、饮食状态、静心练习等，用真实的生活画面代替文字描述。

**提交方式：**
- 上传到 Bilibili / YouTube，把链接贴到 Discussions
- 标题格式：`[视频] 你的主题`

---

### 方式四：图片或手写笔记

手写的修行日记、饮食记录、身体状态图表等。

**提交方式：**
- 图片直接贴到 Discussions 的评论里
- 或新建 Issue，标题格式：`[图片记录] 你的主题`

---

### 方式五：匿名贡献

如果你想分享但不希望实名，可以：
- 在 Discussions 里匿名描述你的经历
- 或发邮件给维护者，由维护者整理后匿名收录（邮件地址见仓库主页）

---

## 贡献者须知

**什么样的记录最有价值：**
- 真实的，包括失败和走偏的部分
- 具体的，不是"感觉好多了"，而是"敲胆经第三周，左腿酸痛减轻，同时发现睡眠变深"
- 有疑问的，你卡住的地方可能正是别人最需要的参照

**什么样的内容不适合收录：**
- 推广特定产品或服务
- 无法验证的医疗声明
- 替代专业医疗建议的内容

**重要提示：**
严重疾病请就医，深度修行请寻访善知识。
这个项目是实践记录和探索，不是医疗建议，不是权威指导。
