## 〇、出发点：为什么现有 SQ 量表不够用

现有三大量表（SQ21 / SISRI-24 / PSI）的共同局限：

| 问题 | 说明 |
|------|------|
| **把"灵性"当能力量** | 默认灵性是可习得的技能或认知能力，可以用 Likert 量表打分。但这与东方传统的根本认知相悖——觉性不是"修出来的"，而是"遮蔽少了之后本来就在的"。 |
| **缺乏身体维度** | 除 PSI 有简略的身体觉知维度外，SQ21 和 SISRI-24 基本是"从脖子往上"的灵性。中医"形与神俱"的整体观没有被纳入。 |
| **无"回归"的结构** | 所有量表都是正向递增的（分数越高越好），没有"做减法"的维度——你放下了多少、停止了什么。而道德经说"为道日损"。 |
| **割裂了见修行** | 测量的是静态截面，没有"见→修→行"的同步反馈结构。 |
| **文化根在西方一神论传统** | 即便是最世俗化的 SISRI-24，其"超验觉知"维度也暗含一个外在超越者（God / Higher Power）的假设，而非"觉性本在"的内在转向。 |

## 一、根本定位：不是"测你多灵"，是"照你走到哪了"

这是量表设计最基础也最容易被忽略的一步——**你到底要测什么？**

### 西方 SQ 量表的逻辑

```
灵商是一种智力 → 智力有高低 → 测出你的分数 → 和常模比 → 你就是这个水平
```

这是一个**比较—排序—标签**的逻辑链。它天然暗示"有的高、有的低"。

### 本方案的逻辑

```
觉性本来就在 → 只是被不同程度的遮蔽 → 
量表帮你照见：此刻你在哪些地方被语言/习惯/执着带走？哪些地方已经松开了？ →
这不是分数，是一面镜子——帮你自己看见，不是替你看见
```

对应《楞严经》卷一阿难的启示：阿难"多闻第一"——智力层面的灵性知识极高——但境界现前照样被带走。按 SQ21 打分他可能极高，但经典说"未全道力"。**知识和状态是两回事。** 本量表要测的是后者。

## 二、维度设计：五层 · 根于经典

从本项目的经典知识库中，提炼五个不可彼此归约的维度。每个维度给出：经典出处（🟢）、操作性定义、与现有 SQ 量表的对应/差异。

### 维度总览

```
                      身（形与神俱）
                           │
              ┌────────────┼────────────┐
              │            │            │
           守（精神内守） 触（六根觉性） 转（回归转换）
              │            │            │
              └────────────┼────────────┘
                           │
                      行（在世而行）
```

### 维度一 · 身 —— 形与神俱（Body-Spirit Unity）

| 项目 | 内容 |
|------|------|
| **经典出处** | 🟢 《黄帝内经·上古天真论》："形与神俱，而尽终其天年"；"食饮有节，起居有常，不妄作劳" |
| **操作性定义** | 一个人对身体信号的觉察程度、身心是否一体运作而非分离、基本生活节律是否合乎自然 |
| **现有 SQ 量表对应** | PSI 的 Body Awareness 维度仅泛泛而谈；SQ21 和 SISRI-24 无此维度 |
| **本量表的独特之处** | 把"身体觉知+生活节律"合为一个基础维度——身体是觉性的锚点，没有身体根基的"灵性"容易沦为空谈 |

**典型条目方向**（非最终条目，仅为设计方向示意）：

- 身体觉察层次：
  - 我能在身体发出疲劳/不适信号时及时停下来（而非用意志硬撑）
  - 我知道自己吃饱了，并能停在刚好
  - 当情绪升起时，我能感知到它在身体哪个位置（胸口紧/胃部不适等）

- 节律层次：
  - 我的饮食有基本规律，不过度也不匮乏
  - 我的睡眠大致跟随昼夜节律
  - 我不会把自己的身体当成完成任务、追求快感的工具去透支

- 形神合一层次：
  - 走路时，我感觉到脚踩在地上
  - 我不会一边吃饭一边刷手机到忘记食物的味道
  - 动手做一件事时（做饭、写字、打扫），身心同在，而不是身体在动、心在别处

**评分方式**：非 Likert 程度量表，采用**频率锚定**："几乎从不 / 偶尔 / 经常 / 几乎总是"。因为测的是"回归身体"这个动作的频率，而非"我相信身心一体"这个信念。

---

### 维度二 · 守 —— 精神内守（Inner Guarding）

| 项目 | 内容 |
|------|------|
| **经典出处** | 🟢 《黄帝内经》："恬淡虚无，真气从之，精神内守，病安从来"；《道德经》第五章："多闻数穷，不若守于中" |
| **操作性定义** | 注意力不向外耗散的倾向，以及内在安定、不被外境轻易掀翻的程度 |
| **现有 SQ 量表对应** | SISRI-24 的 CET（批判性存在思维）是"向外思考存在性问题"，本维度是"向内安住"——方向相反；SQ21 的 Q3（自我掌控）有部分重叠 |
| **本量表的独特之处** | 不是测量你"想"了多少关于灵性/意义的问题（CET 的做法），而是测你"不想"的能力——停下来、收回来、安住在不需要语言的地方 |

**典型条目方向**：

- 安住层次：
  - 独处时，我能安安静静地待着，不需要用手机/娱乐/忙碌来填充
  - 没事可做的时候，我不会立刻感到焦虑或空虚
  - 我能在不说话、不思考"有用的事"的状态中，感到自在

- 不被带走层次：
  - 别人批评我时，内在有一个不被掀翻的稳（而不是立刻防卫或崩溃）
  - 遇到意外变化，我不会习惯性地陷入反复担忧的思维循环
  - 看到让人兴奋的东西（促销/新闻/八卦），我能不被拉着走

- 恬淡层次：
  - 我对"再多一点"的追逐有觉察，能看到自己在追什么
  - 我不需要用"忙"来证明自己有价值
  - 欲望升起来时，我看着它，而不一定跟着它跑

**评分方式**：频率锚定 + 反向题。"总是感到需要填充无聊"在这里是低分——不是因为它不好，而是因为精神外散的频率高。反向题不设"好坏"标签，只描述现象。

### 维度三 · 触 —— 六根觉性（Six-Faculty Awareness）

| 项目 | 内容 |
|------|------|
| **经典出处** | 🟢 《楞严经》卷四："随拔一根，脱黏内伏，伏归元真，发本明耀"；卷六："元明照生所，所立照性亡" |
| **操作性定义** | 在日常生活当下，对六触（眼耳鼻舌身意）的生起、流转、消去有多清晰的看见；以及从"被触带走"到"看见触在转"的转换频次 |
| **现有 SQ 量表对应** | SISRI-24 的 TA（超验觉知）和 CSE（意识状态扩展）部分接近，但偏"特殊体验"而非"日常觉察"；SQ21 Q1/Q2 部分相关，但无"触"的精细概念 |
| **本量表的独特之处** | 这是本量表的核心创新维度——把"觉知"操作化为"对六触生灭的日常觉察"，而非一次性的神秘体验或信念陈述 |

**典型条目方向**：

- 触的看见：
  - 当我因为一句话产生情绪反应时，能"看见"那个反应在身体/心里升起来的那一刻——而不只是事后才意识到
  - 我感觉到一个念头自己升起来、停留、又自己消去（而不一定要把它想完）
  - 吃东西时，我能觉察到味道（舌触）从生起到消失的变化，而不只是机械地吞咽

- 黏/脱的觉察：
  - 我能意识到自己什么时候"黏"在一个念头或情绪上出不来了
  - 被黏住之后，我回到身体感觉或呼吸上的速度
  - 我能在"越想越紧"的苗头出现时就觉察到，而不是想了半小时才发现

- 语言空转的觉察：
  - 我能识别自己什么时候在用更多思考去解决思考带来的痛苦
  - 我能在语言空转刚启动时就"看见"这个循环，而不一定跳进去

**评分方式**：频率锚定 + 情境锚定。不同于维度二的"精神内守"（是一种相对稳定的状态），维度三测的是**动态过程**——你多常"看见"，多快"回来"。因此题目更适合用"发生某情境时，你多常能……"的格式。

### 维度四 · 转 —— 回归与转换（Return & Transformation）

| 项目 | 内容 |
|------|------|
| **经典出处** | 🟢 《道德经》："反者道之动"（回归是道的运动方向）；《黄帝内经》五行情志相胜；《楞严经》："随拔一根，脱黏内伏" |
| **操作性定义** | 从痛苦/卡住的状态中，通过"转换"而非"对抗"来出来的能力倾向——本质是"从系统外部撬开，而非在系统内部加力" |
| **现有 SQ 量表对应** | 无直接对应。这是本量表区别于所有西方 SQ 量表的最根本维度——西方量表只测"正向能力"，不测"如何从负面状态中转换" |
| **本量表的独特之处** | 引入"非对抗性转换"这个东方独有的智慧：不是解决问题，是转换维度；不是更强地对抗，是换一个入口 |

**典型条目方向**：

- 行动转换：
  - 当思绪绕圈停不下来时，我选择去做一件动手的事情（走路/做饭/整理），而不是继续坐着想
  - 情绪卡住时，我能用身体动作来打破僵局

- 触的转换：
  - 一个触让我痛苦时，我能觉察到 "我可以转到另一个触上"（从意触转到身触、从眼触转到耳触）
  - 我不会在让我痛苦的同一个感知通道里死磕

- 情志的自然流转：
  - 我不需要强行压制情绪——它来了、被看见、自然流走
  - 我不会用"正能量"去盖住"负能量"，而是允许情绪被体验和表达完

**评分方式**：频率锚定 + 情境反应描述。"在这种情况下，你更倾向于 A（系统内对抗）还是 B（维度转换）？"——A/B 并非好坏，而是反映当前的主导模式。此维度的评分逻辑是：**不是比谁分高，是帮人看见自己的模式。**

### 维度五 · 行 —— 在世而行（Living It in the World）

| 项目 | 内容 |
|------|------|
| **经典出处** | 🟢 本项目八字心法："快乐·动手·开源·分享"；《道德经》第五："虚而不屈，动而愈出"——虚静之后是"动而愈出"，回归之后是再出发 |
| **操作性定义** | 觉知有没有落到实际生活里——是不是只在坐垫上有、一起身就没了；以及是否从"自我关怀"扩展到了"对他人和世界的打开" |
| **现有 SQ 量表对应** | SQ21 Q4（社会掌控/灵性临在）有重叠，但偏"成为导师/改变推动者"的向外主导角色；本维度更强调"回归之后再出发"的循环——服务是流动的副产品，不是另一个 KPI |
| **本量表的独特之处** | 把"行动"放到"回归"之后——回归→带着清明再出发，而不是直接从"自我觉知"跳到"改变世界" |

**典型条目方向**：

- 快乐的检验：
  - 回顾最近一周，有没有"不需要理由就感到轻松"的时刻
  - 做事的动力更多来自内在的自然流动，还是来自"应该/焦虑/要证明自己"

- 动静循环：
  - 我能安静下来，也能带着那份安静去行动（而不是安静久了萎靡,行动久了焦躁）
  - 做完一件事后，我能不能自然地休息，而不是立刻跳到下一件

- 打开自己：
  - 我愿不愿意把自己摸索到的东西分享出去——包括走过的弯路
  - 我对"别人会不会超过我"或"别人怎么看我"的担忧，在多大程度上限制了分享
  - 拥抱与付出：你对妻子/先生的拥抱出于什么（习惯？需要？还是自然地分享爱？）——这是最难的一层

- 服务的自然性：
  - 帮助别人时，我是"在付出"（期待回报或认可），还是自然地流淌
  - 我不需要通过"帮助别人"来确认自己有价值

**评分方式**：频率锚定 + 情境选择题。此维度最容易产生"社会期待效应"（答题者选"应该的"而非"真实的"），因此需要设计一些**间接情境题**来降低防御。例如不直接问"你是否乐于分享"，而是描述一个具体情境让答题者选择自己最可能怎么做。

## 三、条目设计方法论

### 3.1 关键挑战：东方语境下，如何避免"越测越偏"

传统 Likert 量表隐含一个假设：**"越多越好"**。「我经常冥想」→高分，「我从不冥想」→低分。但在东方传统里，这个逻辑不一定对：

- "我每天都坚持打坐两小时"——如果这是用意志力在硬撑、在自我对抗，那在维度二（精神内守）上反而可能低分。
- "我对灵性完全没有概念"——如果是自然天真、不立"所"的状态，在维度三（六根觉性）上反而是接近"元明"的。

**因此条目设计须遵循的几条原则**：

1. **不预设"多做=高分"**。"冥想频率"这类题只有当行为是自然流出而非意志强迫时，才反映真实状态。
2. **多用现象描述，少用信念陈述**。不用"我相信万物互联"，用"我在自然中感到……"——因为信念是可以背出来的，现象无法伪装。
3. **设反向情境题**。不直接问"你觉知力如何"，而是问"你多常因为走神而撞到门框/忘带钥匙/吃到撑了才发现"——通过行为痕迹来反向推断。
4. **引入"不给糖"原则**。题目不暗示任何"正确答案"，不给答题者一种"选这个比较好"的糖。某些维度甚至可以故意设置两个看起来都合理、但反映不同状态的选项。

### 3.2 条目格式建议

混合使用三种格式，以降低单一格式的系统偏差：

| 格式 | 适用场景 | 示例格式 |
|------|----------|----------|
| **频率锚定** | 可计数的日常行为/觉察频次 | "几乎从不—偶尔—经常—几乎总是"（4 点，去掉中性以迫使选择） |
| **情境选择题** | 反应模式（对抗 vs 转换、黏住 vs 松开） | 描述一个具体情境，给两个或多个行为倾向选项 |
| **体验回忆** | 觉知的"事后觉察" | "在过去一周里，你______的情况发生过几次？" |

### 3.3 避免的条目类型（基于 SISRI-24 的教训）

| 避免 | 原因 | 替换方向 |
|------|------|----------|
| "我经常思考生命的意义" | 思考≠觉知；思考过多反而可能是语言空转 | "当你不再想'生命的意义是什么'时，日常生活本身能否给你足够的充实感" |
| "我能与更高的力量连接" | 暗含有神论框架，与佛道"觉性本在"不兼容 | "在安静的时刻，你是否感到一种不需要语言描述的……在场" |
| "我愿意帮助他人" | 社会期待效应太强 | 描述一个需要牺牲便利去帮人的具体情境，问"你更可能怎么做" |

## 四、评分与解读：不做排名，做镜子

### 4.1 不做"总分"

一个总分（如 SISRI-24 的 0-96）在东方框架下没有意义。因为：

- 五个维度不是累加的——不是"你身体好+觉知好+会转换=你修得好"。
- 一个人的剖面才是有意义的：有的维度很强、有的维度还没开始，这是正常的。
- 维度之间可能有关联（身体基础不牢 → 六根觉性难以稳定），但不是线性因果。

### 4.2 五维剖面图（Radar Profile）

用雷达图展示五个维度的相对位置。不标记"常模线"（没有"正常人"作为参照），只标记作答者自己的：

```
        身
        /\
       /  \
   行 /    \ 守
     |      |
     |      |
   转 \    / 触
       \  /
        \/
```

### 4.3 解读文字的"镜子"原则

解读不是告诉对方"你这里做得不好，要加油"。而是**照见**：

> 你在「身」这个维度上的描述，呈现出身体信号被忽略的频率较高——这不代表"不好"，只是说这里有一块你可能还没注意到的地方。当你准备好了，可以试试在一天里挑一件事（比如吃饭），全程只做这一件事。

（这保持了本项目"不给糖"的原则——不塞答案，只帮人看见，然后让他自己走过去。）

## 五、开发与验证路径

### 5.1 阶段一：条目池生成（本项目可做的事）

1. 从五维度定义出发，每个维度生成 15~20 条候选条目
2. 条目须经"两轮自检"（照 CLAUDE.md ④节）：准确性 + 边界
3. 每条目标注经典出处层（🟢经典 / 🟡实践经验 / 🔴一家之言）

### 5.2 阶段二：专家内容效度

1. 邀请三位以上对中医+佛道经典有实修经验、同时对心理测量有了解的人
2. 逐条评审：条目是否归属于它要测的维度？有无误导性？有无"给糖"？
3. 计算 CVI（内容效度指数），0.78 以下条目删除或修改

### 5.3 阶段三：小样本试测（N ≈ 30-50）

1. 检验条目理解度——有没有人读不懂题目
2. 检验区分度——是否所有答题者都选同一个答案（天花板/地板效应）
3. 首次探索性因子分析（EFA）——条目是否大致按五维聚类

### 5.4 阶段四：验证性研究（N ≥ 300）

1. CFA 验证五因子结构
2. 内部一致性（α > 0.70 每维度）
3. 与 SISRI-24 的聚合/区分效度：本量表某些维度应与 SISRI-24 有中等相关（如"触"与 CET/TA），某些应无相关（如"身"与 CET），以证明测的是不同东西
4. 与正念量表（FFMQ）、心理幸福感量表的效标关联
5. 重测信度（间隔 2-4 周）

### 5.5 阶段五：文化适配

核心问题：这个量表在非华语/非佛道文化背景中是否有效？

回译（back-translation）的挑战：某些概念在英语中没有对等词——"转触""恬淡""守中"不能直译。需要在每个目标文化中做**概念对等的本地化**（而非字面翻译），然后重新做效度验证。

## 七、风险与坦诚的局限

1. **这是一个理论构想，未经实证。** 五因子结构可能在 EFA/CFA 中不被支持，维度之间可能高度相关而需要合并。
2. **把"觉性"操作化为量表条目本身是一个悖论：** 越精细地去测量"不立所"的状态，越容易让答题者"立所"——在答题那一瞬间，觉性就被语言盖住了。这是任何纸笔量表都绕不开的根本局限。
3. **文化边界：** 本量表根植于中医和儒释道框架。将其用于不同文化背景的人群时，条目可能在概念层面就不成立（比如"六触"的概念在西方心理学中没有对应）。
4. **动机偏差：** 如果用于组织选拔或灵性领导力评估（如 SQ21 的市场定位），会产生"装好"的强烈动机。本量表的初衷是**自用·照见**，不是**他用·选拔**。
5. **不替代实修：** 量表是地图，不是路。填完量表不会让人变得更觉知——觉知只能自己回到。这个量表只是帮你看看：此刻，你在哪里。
