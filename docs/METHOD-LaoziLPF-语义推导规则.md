# LaoziLPF 语义推导规则 · Semantic Derivation Rules (SDR)
# 从逐字训释到白话译文,每一步允许什么、禁止什么

> 与 [`METHOD-LaoziLPF-四层译文推导.md`](METHOD-LaoziLPF-四层译文推导.md) 配套:那份定义"四层输出"总框架,本文档定义 **Layer 2(语义)→Layer 3(功能)** 之间每步的操作边界。
> 方法论公开版,不含源项目版权引文。整理：2026-07-21。

---

## 1. 逐字训释规则（Layer 1 → Layer 2）

1. **来源优先级**:解释者点评 > 解释者注疏 > 解释者勘误说明 > 本稿按其方法推演。每条注释必须标明属于哪一级,不得混同（把"本稿推演"写成"原书实录"是最严重的诚实性违规）。
2. **知识对象（EO）优先**:字若已有登记在案的确认训释（Evidence Object,含出处/引文/置信度）,直接引用,不重新论证;没有的按白名单规则判"功能义/语法义",判不出就标低信心 + 挂"未决",**不许硬凑**。
3. **同字异位不同训**:允许,但每次判定必须写出判据与对照条目（"与某章某注同字异位不同训,因…"）。
4. **字义判定的封闭性**:Layer 2 只能使用所选解释体系的材料 + 《说文》/段注一系字书;**任何外部体系（禅宗、静心传统、现代心理学…）的说法都不得参与字义判定**——那是 Layer 4 的事。这是 Layer 2 与 Layer 4 之间的硬边界。
5. **借字通则**:借字须循所记之词取义,不循字形本义（例:"而后"之"后"记的是先后字{後},拿"后=继体君"的本义硬套就是错——这是源项目里真实发生并撤销过的推演）。

## 2. 语义组合规则（Layer 2 → Layer 3）

1. **先功能后逻辑**:先组合功能短语,再处理句法逻辑,两步不可颠倒。
2. **同构推演**:本句无原译、但同章有结构相同的句子已有原译时,允许按同构推演,必须标明"依句 N 同构推演"+ 中等信心,不得标高。
3. **推演封闭性**:功能/逻辑关系的用词必须能逐一回指 Layer 2 已定字义,不得为"更通顺"引入新概念、新读法。
4. **照录优先**:解释体系原书有整句译文的直接照录标"实录",不重新推演;实录与已有推演冲突时,**撤销推演、改采实录、留订正记录**（撤销痕迹永不删除——这既是诚实,也是方法论进化的原始数据）。

## 3. 现代语言表达规则（Layer 3 → 白话）

1. **转写不是翻译**:白话句的每个词都要能在 Layer 3 找到来源。
2. **可以现代化措辞,不能现代化概念**:换更常见的说法可以,但不能替换成外来概念（如把体系专有的修炼术语换成"冥想"这类不完全对应的外来词——概念替换会模糊本训与外部概念的边界）。
3. **专有术语保留可追溯性**:体系专有读法首次出现保留原词,让读者能对照注释查证。

## 4. 译文校验清单（每句完成后必查,任一不过即打回）

1. 白话译文中的每个概念,能否在逐字注释里找到对应字头？（覆盖自检）
2. 是否与同章/他章已确立的训释一致（没有"同字异位"却读法不同的情况）？
3. 是否存在把推演写成实录、或把实录简化到丢失关键限定词？
4. 高频关键字的白话是否写出了训释本义而非只挂编号？（防"绕行字典"——只写代号不写含义,机器与读者都无法核查）

## 5. 与 Layer 4 的边界

Layer 2/3 的任何一步都不得引用实践对照层的资料。**某字在解释体系里确实找不到依据时,唯一正确的做法是标"未决"或低信心,而不是用外部资料"补齐"**——解释体系自成一体,混入外部体系会让单一事实源（SSoT）崩溃,这是源项目从立项起坚持的第一红线。
